ICD360S e.V. Vorsitzer — Flatpak Repository

Auto-Update für die Vorsitzer-App. Einmal hinzufügen, danach läuft flatpak update automatisch beim nächsten Release.

Installation

flatpak remote-add --user --if-not-exists icd360s-vorsitzer \
  https://icd360s-e-v.github.io/vorsitzer/icd360s-vorsitzer.flatpakrepo
flatpak install --user icd360s-vorsitzer de.icd360s.Vorsitzer

Update

flatpak update --user
